- Derivar cenários de teste dos critérios de aceite e do escopo consolidado.

- Registrar lacunas remanescentes.

# Saída esperada
1. Testes criados ou ajustados.
2. Cenários cobertos.
3. Resultado da execução.
4. Lacunas restantes.
